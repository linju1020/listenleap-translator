# ListenLeap 翻译助手

一款为英语学习者打造的Chrome扩展插件，集成在ListenLeap网站上使用。

## 功能特性

### 1. 句子翻译
在每句英文下方自动显示中文翻译，方便理解课程内容。

### 2. 单词查词
点击任意英文单词，在右侧弹出Drawer显示词典详情：
- 英文单词 + 音标
- 词性 + 英文释义
- 例句
- 异步加载中文翻译

### 3. 发音功能
- 点击🔊按钮播放英文发音（使用有道词典音频）
- 播放时图标有动画效果

### 4. 生词本
收藏需要记忆的单词：
- 点击♥收藏单词
- 查看已收藏的单词列表
- 删除不需要的单词

### 5. 循环播放
在生词本中支持循环播放：
- 勾选"朗读中文"可同时播放中文解释
- 最多支持50遍
- 播放时点击任意位置或切换Tab自动停止

### 6. 拖动排序
可以拖动调整生词本中单词的顺序。

## 使用的API服务

| 服务 | 用途 |
|------|------|
| Free Dictionary API (dictionaryapi.dev) | 英语词典释义 |
| Google Translate | 中英文翻译 |
| 有道词典 | 英文发音 |
| Web Speech API | 中文朗读 |
| Font Awesome | 图标 |

## 安装方法

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `listenleap-translator` 文件夹

## 使用方法

1. 访问 ListenLeap 网站的课程页面
2. 等待页面加载，英文句子下方会自动显示中文翻译
3. 点击任意单词，右侧弹出词典详情
4. 点击♥收藏单词到生词本
5. 切换到「我的生词」Tab查看和管理生词
6. 使用循环播放功能学习生词
